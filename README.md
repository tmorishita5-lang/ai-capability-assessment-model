# AI Capability Assessment Model (Morishita Model, 2025)

This repository presents the **AI Capability Assessment Model**, developed by Takashi Morishita (2025).

The model explains why AI outputs differ between individuals, and why user capability plays a central role in determining AI effectiveness.

---

## Core Formula

**Output = f(User Capability) × g(Operational Skill) × Environment**

- **f（認知能力）**：抽象化・判断・専門知識  
- **g（操作能力）**：プロンプト設計・反復・制約理解  
- **Environment**：AIモデル特性・ツール・設定
